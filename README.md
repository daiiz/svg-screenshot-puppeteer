# svg-screenshot-puppeteer

## Bookmarklet
- https://scrapbox.io/daiiz/puppeteer-svg-screenshot撮影コマンドを生成するbookmarklet

## Run
```
$ babel-node index.js -u https://scrapbox.io/daiiz/The_Great_Burgerに行ってきた -r 0,0,687,687 -v 1054,667
```
